# Reviewer Checklist

Bu değişikliği dış denetleyici olarak kontrol et.

## Kontrol Maddeleri

1. İstek gerçekten karşılanmış mı?
2. Scope gereksiz büyümüş mü?
3. Var olan çalışan sistem bozulmuş mu?
4. Uydurma endpoint, repo, dosya adı veya teknik iddia var mı?
5. Dosya değişiklikleri minimum ve hedef odaklı mı?
6. Test/çalıştırma sonucu yeterli mi?
7. Commit'e hazır mı?

## Çıktı Formatı

- Verdict: PASS / PARTIAL / FAIL
- Correct Findings
- Risks
- Missing Checks
- Commit Ready: YES / NO
