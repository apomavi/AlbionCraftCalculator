2. AODP developer sayfası
3. AODP items / identifier yapısı
4. ao-bin-dumps GitHub reposu ve içindeki işimize yarayacak dosyalar
5. city bonuses
6. focus sistemi
7. premium etkileri
8. market tax / setup fee / satış vergileri
9. craft recipe kaynakları
