# DEC-0001 — Request ve Run Kayıt Standardı

- Tarih: 2026-04-09
- Durum: Accepted

## Bağlam

Projede hem `input/request.md` hem de `requests/REQ-0001.md` kullanımı vardı.
Bu durum append-only kayıt kuralını zayıflatıyordu.

## Karar

1. Ana request girişi sadece `requests/REQ-xxxx.md` olacak.
2. `input/request.md` şimdilik silinmeyecek, sadece deprecated/pasif kalacak.
3. Kalıcı run kayıtları sadece `reports/RUN-xxxx/` altında tutulacak.
4. `output/` klasörü geçici çalışma alanı olarak kalacak.
5. `runs/index.csv` append-only run kaydı olarak güncellenecek.
6. `data/manifests/` veri import manifestleri için ayrılacak.

## Sonuç

Request dosyaları ezilmeden saklanacak, her run yeni klasöre yazılacak ve kayıt zinciri geriye dönük izlenebilir olacak.
